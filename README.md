# Juris
Juris é uma plataforma web desenvolvida como Projeto de Conclusão de Curso (TCC), projetada para otimizar o Gerenciamento de Processos Jurídicos.

O sistema foca especificamente nas necessidades de escritórios de pequeno porte e da jovem advocacia, oferecendo uma solução acessível e eficiente para organização de rotinas legais.
